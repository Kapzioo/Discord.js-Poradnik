# Discord.js-Poradnik
Jak zrobić swojego pierwszego prostego bota!
Pobieramy to repozytorium a następnie je wypakowujemy na pulpit!
Następnie uzupełniamy plik config.json!
Potem w terminalu bota wpisujemy npm install
A kiedy już nam się zakończy ta funkcja wpisujemy w terminalu node index.js lub node .
